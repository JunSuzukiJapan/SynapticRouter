# تحليل التوجيه في نمذجة اللغة عبر المجالات (كود / رياضيات / نص)

## نظرة عامة
أجرينا تجربة مع SRA لتعلم ثلاثة مجالات (الكود، الرياضيات، النص) بمفردات مختلفة في وقت واحد.

## جودة الاستدلال
بعد 1000 خطوة، تم تحقيق تنبؤات دقيقة دون تداخل.

## تحليل التخصص
1. **التجانس الأولي**: في البداية، استخدمت المهام جميع المشابك الـ 16 بشكل موحد.
2. **التخصص**: انفصل التوجيه بوضوح.
   - استخدم `Code` **المشبك 8**.
   - استخدم `Math` **المشابك 10 و 13**.
   - استخدم `Text` **المشابك 0 و 15**.

## استنتاج
اكتشف النموذج بشكل مستقل "استخدام المشابك المناسبة لكل مهمة"، مما منع النسيان الكارثي.
