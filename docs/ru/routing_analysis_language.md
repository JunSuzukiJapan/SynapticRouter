# Анализ маршрутизации в кросс-доменном языковом моделировании (Код / Математика / Текст)

## Обзор
Мы провели эксперимент с использованием SRA для одновременного обучения трем доменам (код, математика, текст) с совершенно разными словарями.

## Оценка качества вывода
После 1000 шагов были достигнуты точные предсказания без интерференции.

## Анализ специализации синапсов
1. **Гомогенизация в начале**: Сначала каждая задача равномерно использовала 16 синапсов.
2. **Специализация**: Позже маршрутизация разделилась.
   - `Code` использовал **Синапс 8**.
   - `Math` использовал **Синапсы 10 и 13**.
   - `Text` использовал **Синапсы 0 и 15**.

## Заключение
SRA-модель автономно открыла "использование правильных синапсов для каждой задачи", предотвращая катастрофическое забывание.
